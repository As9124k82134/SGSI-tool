## Setup
```
GitHub:  
git clone --recurse-submodules https://github.com/rahulkhatri137/SGSIs -b main
cd SGSIs
su
chmod -R 777 ./
./setup.sh  
```

## Build SGSI:
```
sudo bash url2SGSI.sh <URL> <TYPE>:<NAME>
```

